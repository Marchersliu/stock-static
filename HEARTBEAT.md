## 🔍 每日数据核查（股票监控看板）

### 核查时间
- **交易日 15:30**：Tushare发布当日日线后 → 运行核查脚本
- **非交易日**：跳过（使用最后交易日的静态数据）

### 核查命令
```bash
cd /Users/hf/.kimi_openclaw/workspace && python3 verify_dashboard_data.py
```

### 核查内容
1. **持仓总览表**（5只）：现价、日涨跌、3日/5日/7日涨跌
2. **计划建仓表**（8只）：现价、日涨跌、3日/5日/7日涨跌
3. **三因子标识**：消息面/资金面/技术面（后端启动时动态计算）
4. **📰 盘前新闻**（PREMARKET_NEWS数组）：财务数据真实性、日期合理性、过时检测
5. **🏭 产业链表格**：标记为"⚪ AI估算/行业均值，需人工定期复核"

### 新闻核查规则（新增）
- **财务数据**：提取新闻中的营收/净利润数字，与Tushare `pro.income()` 自动比对
- **日期检查**：新闻日期不能为未来日期，超过7天自动标记"可能已过时"
- **不可验证关键词**：含「将/拟/计划/在手订单/意向协议」的新闻自动标记⚠️，需人工确认来源
- **错误阈值**：营收偏差>10%、净利润偏差>15% → 立即输出 ❌ 并修正

### 核查规则
- 日涨跌/3日/5日/7日涨跌：必须与Tushare Pro `pro.daily()` 严格一致
- 非交易日：使用 `pro.trade_cal()` 查找最后一个交易日，与HTML静态值对比
- 允许误差：±0.02%（四舍五入）
- 任何差异立即输出 ❌ 并修正 `stock_dashboard.html` 静态值

### 防错铁律
1. **绝不再犯**：给表格加新列时，不能直接复制前一列的值
2. **每次修改后**：运行 `verify_dashboard_data.py` 确认一致
3. **修改步骤**：Tushare拉取 → 计算 → 填入HTML → 运行核查 → 确认通过 → 同步iCloud

## 📈 九州一轨（688485）每日深度分析

### 分析时间
- **交易日收盘后**（16:00~16:30）：生成当日深度评估报告
- **持仓期间持续执行**，直至清仓或用户取消

### 分析维度（七层穿透）

#### 1. 技术面（价格行为）
- RSI(6/12/24)、KDJ(9,3,3)、MACD(12,26,9) 三线状态
- MA(5/10/20/60) 排列与偏离度
- 布林带(BOLL) 位置、收口/开口趋势
- 支撑位/压力位（前高/前低/密集成交区）
- 换手率、量比、振幅、成交量趋势
- 量价关系：放量上涨/缩量回调/背离信号

#### 2. 资金面（主力行为）
- 超大单/大单/中单/小单 四维资金流向
- 融资余额变化率、融券余量、担保品比例
- 机构持仓变动（基金季报/社保/QFII/北向资金）
- 大宗交易（折价率、对手方、关联关系）
- 龙虎榜（营业部席位、游资动向）

#### 3. 消息面（信息驱动）
- 当日公告：业绩、并购、股权变动、增减持
- 行业新闻：轨交投资政策、半导体设备国产化
- 概念关联：金刚石、光模块、半导体材料热度
- 监管动态：问询函、关注函、异常波动
- 舆情情绪：社交媒体讨论热度、机构研报观点

#### 4. 基本面（价值锚定）
- 营收/净利/毛利率/ROE 季度环比与同比
- 半导体转型进展：晶禧并表时点、业绩承诺达成率
- 合资公司通创九州金刚石：注册资本到位、订单获取
- 同行对比：时代新材(600458)、天铁股份(300587) 估值/增速
- 估值模型：PB/PS/EV/EBITDA、DCF情景分析

#### 5. 产业链（位置与格局）
- 上游：特种钢材/橡胶/芯片供应价格与产能
- 下游：国铁/城轨投资规划、招标节奏
- 半导体金刚石：市场规模、竞争对手（黄河旋风、中兵红箭）
- 轨交减振：行业集中度、价格战风险

#### 6. 风险矩阵（概率×影响）
| 风险类型 | 概率 | 影响 | 监控指标 |
|---------|------|------|---------|
| 并购整合失败 | 中 | 极高 | 晶禧季度业绩、商誉减值 |
| 估值回归 | 高 | 高 | PB>8/股价脱离MA20>50% |
| 主业持续恶化 | 中 | 高 | 钢弹簧收入同比降幅 |
| 政策风险 | 低 | 中 | 国铁投资预算变化 |
| 大股东减持 | 低 | 中 | 增减持公告 |

#### 7. 操作建议（动态评估）
- 目标价区间（第一/第二目标）与达成概率
- 止损线动态调整（基于ATR波动率）
- 仓位建议（加仓/减仓/持有的触发条件）
- 关键节点日历（季报发布、股东大会、解禁日）

- **PDF版本**：浏览器打印生成，同步到iCloud

## 股票实时盯盘监控（交易日盘中6次 + 收盘1次）

### 盘中检查时间点
1. **9:35** 开盘检查：开盘价、集合竞价、早盘资金流向、隔夜消息
2. **10:30** 盘中检查：实时价格、成交量、涨跌幅、±7%异动
3. **11:25** 午前检查：上午收盘前最后检查，上午走势特征记录
4. **13:05** 午后开盘：下午开盘首波走势、大单异动
5. **14:30** 尾盘检查：大资金操作时点、急拉急跌、收盘前主力动向
6. **16:00** 收盘总结：全天数据汇总、技术面更新、消息面整理、明日策略

### 触发提醒的异常条件
- 持仓股单日涨跌幅 ≥ ±7%
- 触及目标价（九州一轨60/65，宝丰29建仓，华友62-64建仓等）
- 触及止损线（九州一轨49，迈瑞140等）
- 主力资金异常流入/流出（九州一轨单日主力±5000万+）
- 原材料价格重大变化（锂/钴/镍/锶/原油单日±5%）
- 持仓股突发公告、龙虎榜、大宗交易

### 静默规则
- 无异常 → 回复 HEARTBEAT_OK（不打扰）
- 有异常 → 推送提醒（含价格、原因、建议操作）

### 监控页面
