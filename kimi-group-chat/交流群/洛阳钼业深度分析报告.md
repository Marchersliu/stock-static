> 报告日期：2026-05-04 | 当前股价：约18.62元 | 市值：约3,980亿元

## 一、公司概况

洛阳栾川钼业集团股份有限公司（603993.SH / 03993.HK），成立于1969年，2012年A股上市，是全球领先的多元化矿业巨头。业务覆盖**铜、钴、钼、钨、铌、磷、黄金**七大金属品类，运营网络横跨亚洲、非洲、南美洲。

**核心定位**：从"单一铜钴矿商"向"多金属全球资源巨头"演进，打造"铜+黄金"双轮驱动战略。

## 三、分业务板块深度分析

### 3.1 铜业务（核心基本盘，毛利占比61%）

| 指标 | 2025年 | 2026Q1 |
|------|--------|--------|
| 产量 | 74.1万吨 | 18.8万吨 |
| 销量 | - | 18.2万吨（同比+47%） |
| 毛利率 | 55.16% | 63.69% |
| 收入 | 551亿元 | - |

**成长驱动**：
- KFM二期预计**2027年投产**，新增10万吨/年产能
- TFM三期规划中，目标**2028年铜产量100万吨**
- LME A级注册认证通过，打通国际交割通道
- 全球铜供需缺口25-60万吨，价格中枢上移

**敏感性**：铜价每下跌10%，2026年净利润减少约**31亿元**。

### 3.2 钴业务（弹性品种，量缩价升）

| 指标 | 2025年 | 2026Q1 |
|------|--------|--------|
| 产量 | 11.75万吨 | 3.05万吨 |
| 销量 | - | 0.20万吨（同比-91.8%） |
| 毛利率 | 63.62% | 86.27% |

**关键变量**：
- 刚果(金)钴出口配额制：2026-2027年配额9.66万吨/年，较2024年削减**48%**
- 全球钴市场首次出现持续性短缺（缺口0.6-2.5万吨）
- MB标准级钴价已从$10/lb反弹至**$25-26/lb**
- 敏感性：钴价每上涨$5/lb，增厚利润约**10亿元**
- **风险**：数万吨钴库存悬在账上，既是利润蓄水池也是风险敞口

### 3.3 钨业务（季度黑马，战略金属）

- 2026Q1营收19.6亿元（**同比+292.9%**）
- 毛利率70.93%
- 全球钨供需紧平衡，中国出口管制收紧，钨价创历史新高
- 预计2026年钨收入从24亿跃升至**64亿元**（+163%）

### 3.4 黄金业务（新增长极）

- 2026年1月完成巴西金矿交割（10.15亿美元）
- 2026年预计产金**6-8吨**
- 2029年远期黄金产能目标**20吨**
- 厄瓜多尔奥丁矿业前期规划中

## 七、风险因素

### 7.1 金属价格波动风险（最高优先级）
- 铜价每跌10% → 净利润减31亿元（-10.3%）
- 钴价波动剧烈，库存减值风险

### 7.2 地缘政治与政策风险
- 刚果(金)贡献约75%利润，政治环境不稳定
- 钴出口配额制增加销售不确定性
- 超额利润税、矿业法规调整风险

### 7.3 财务风险
- 短期借款从271亿升至326亿元
- 发行12亿美元可转债（2027年到期）
- 汇率波动：Q1外币折算损失13.6亿元

### 7.4 项目执行风险
- KFM二期、TFM三期建设进度
- 巴西金矿整合运营（刚交割，存在法律纠纷）
- 刚果(金)电力、物流基础设施瓶颈

### 7.5 行业风险
- 钴：电池无钴化趋势长期压制需求
- 铜：全球经济衰退导致需求下滑

## 八、核心投资逻辑总结

### 看多逻辑
1. **铜业务量价齐升**：全球缺口+产能扩张，毛利率大幅提升至64%
2. **钴的期权价值**：配额制推动价格上行，$25/lb已较底部翻倍
3. **黄金新叙事**：2026年首次并表，远期20吨产能
4. **钨的战略溢价**：出口管制+供需紧平衡，价格创历史新高
5. **估值偏低**：PE 12倍，机构目标价对应30-50%上行空间
6. **高分红**：承诺2024-2026年分红比例不低于净利润40%

### 看空/谨慎逻辑
1. 前期涨幅巨大（52周低点5.65元→当前18.62元，已涨230%）
2. 钴库存积压风险（数万吨待售）
3. 短期债务压力+可转债到期
4. 刚果(金)地缘政治不确定性
5. 铜价已处历史高位，回调风险
