# 认知记忆模型理论

## 人脑记忆的分层架构

```
感觉记忆 (Sensory)     → 毫秒级，原始输入
    ↓
工作记忆 (Working)     → 秒级，当前任务上下文
    ↓
短期记忆 (Short-term)  → 分钟~小时，临时存储
    ↓ 巩固(Consolidation)
长期记忆 (Long-term)   → 天~年，永久存储
    ├── 情境记忆 (Episodic)  → 我做了什么
    ├── 语义记忆 (Semantic)  → 我知道什么
    └── 程序记忆 (Procedural) → 我会做什么
```

## 核心机制

### 1. 编码 (Encoding)

5W1H情境编码确保记忆有完整上下文，而不只是孤立事实：
- **When**: 时间锚点，用于遗忘曲线计算
- **Where**: 空间/文件锚点，用于情境回忆
- **Who**: 社会锚点，涉及的人或实体
- **What**: 内容核心
- **Why**: 因果链，理解动机
- **How**: 执行路径，程序提取

### 2. 联想网络 (Associative Network)

人脑回忆不是搜索，而是激活：
- 想到"苹果"→自动激活"水果""红色""甜味""乔布斯"
- 想到"sed"→自动激活"危险""HTML破坏""edit替代"

**扩散激活模型**:
```
激活强度 = 初始强度 × 关联强度^深度
```

### 3. 遗忘曲线 (Forgetting Curve)

Ebbinghaus 发现：记忆随时间指数衰减
- 20分钟后遗忘 42%
- 1小时后遗忘 56%
- 1天后遗忘 74%
- 1周后遗忘 77%

**系统实现**:
```
strength = base × e^(-decay × days)
每次访问: base += 0.1 (上限3.0)
情感权重: decay *= (1 - emotional × 0.5)
```

### 4. 巩固 (Consolidation)

睡眠中，海马体将短期记忆转入皮层长期存储。

**系统模拟**:
- Session结束 → 扫描日志 → 提取事件 → 编码
- 24小时后再次访问 → 标记为"已巩固"
- 未巩固的记忆更容易遗忘

## 与传统记忆系统的区别

| 维度 | 传统系统 (MEMORY.md) | 认知记忆 (cognitive-memory) |
|------|---------------------|---------------------------|
| 存储 | 线性文本 | 结构化网络 |
| 检索 | 关键词grep | 联想扩散激活 |
| 遗忘 | 永久保存 | 强度衰减 |
| 关联 | 无 | 概念网络 |
| 情境 | 无 | 5W1H完整编码 |
| 程序 | 无 | 步骤链条 |
| 情感 | 无 | 重要性权重 |

## 使用建议

1. **高频概念** = 高频访问的节点，联想网络中心
2. **情感标记** = critical的记忆几乎不会遗忘
3. **定期巩固** = 每天运行一次 consolidation.py
4. **主动回忆** = 遇到问题时先用 --associate 联想，而不是 --search 搜索
