# 中文去AI化技能

## AI痕迹特征

| 特征 | 示例 | 修改方向 |
|------|------|---------|
| 过于工整的排比 | "首先...其次...最后..." | 打散结构，改用自然过渡 |
| 高频连接词 | "值得注意的是""综上所述" | 减少或替换为口语化表达 |
| 绝对化表述 | "必然""绝对""毫无疑问" | 加入限定词"可能""大概率" |
| 数据堆砌 | 连续3+数字无解读 | 每个数字后加一句人话解释 |
| 缺少个人语气 | 全程客观无立场 | 适当加入"我认为""说实话" |
| 段落过长 | 200+字不分段 | 拆成100字以内短段 |

## 处理流程

1. **句式打散** → 长句拆短，排比改散句
2. **用词替换** → AI高频词→自然口语词
3. **节奏调整** → 加入设问、感叹、口语停顿
4. **个性化注入** → 加入主观判断、情绪色彩
5. **段落重构** → 长短交替，避免机械均匀

## 快速使用

```bash
cd /Users/hf/.kimi_openclaw/workspace/skills/humanizer-zh/scripts
python3 humanize.py input.md --output humanized.md --intensity medium
```

## 强度等级

- `light`：轻度调整，保留原结构，替换高频词
- `medium`：中度调整，打散排比，加入口语化表达
- `heavy`：重度调整，重构段落，加入个人风格
