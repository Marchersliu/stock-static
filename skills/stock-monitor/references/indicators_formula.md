# 技术指标计算公式

## KDJ 随机指标

**参数**：N=9, M1=3, M2=3

```
RSV = (今日收盘价 - N日内最低价) / (N日内最高价 - N日内最低价) × 100

K值 = 2/3 × 前日K值 + 1/3 × 当日RSV
D值 = 2/3 × 前日D值 + 1/3 × 当日K值
J值 = 3 × 当日K值 - 2 × 当日D值
```

**初始值**：K=D=50（或RSV）

**信号**：
- K > D 且 K < 20：超卖区，买入信号
- K < D 且 K > 80：超买区，卖出信号
- K上穿D（金叉）：买入信号
- K下穿D（死叉）：卖出信号

---

## MACD 指数平滑异同平均线

**参数**：SHORT=12, LONG=26, MID=9

```
EMA(12) = 前日EMA(12) × 11/13 + 今日收盘价 × 2/13
EMA(26) = 前日EMA(26) × 25/27 + 今日收盘价 × 2/27

DIF = EMA(12) - EMA(26)
DEA = 前日DEA × 8/10 + 今日DIF × 2/10
MACD柱 = (DIF - DEA) × 2
```

**信号**：
- DIF上穿DEA（金叉）：买入信号
- DIF下穿DEA（死叉）：卖出信号
- MACD柱由负转正：多头转强
- MACD柱由正转负：空头转强

---

## RSI 相对强弱指标

**参数**：N=6, 12, 24

```
RS = N日内上涨日平均涨幅 / N日内下跌日平均跌幅
RSI = 100 - 100 / (1 + RS)
```

**信号**：
- RSI > 80：超买，卖出信号
- RSI < 20：超卖，买入信号
- RSI 50附近：观望

---

## 移动平均线 MA

**公式**：
```
MA(N) = (最近N日收盘价之和) / N
```

**常用周期**：5日、10日、20日、60日、120日、250日

**信号**：
- 5日上穿10日（金叉）：短期买入
- 5日下穿10日（死叉）：短期卖出
- 多头排列（5>10>20>60）：强势上涨
- 空头排列（5<10<20<60）：弱势下跌

---

## 布林带 BOLL

**参数**：N=20, K=2

```
中轨 = MA(20)
标准差 = 最近20日收盘价的标准差
上轨 = 中轨 + 2 × 标准差
下轨 = 中轨 - 2 × 标准差
```

**信号**：
- 股价突破上轨：超买，可能回调
- 股价跌破下轨：超卖，可能反弹
- 布林带收窄：即将变盘
- 布林带扩张：趋势加速

---

## 综合买卖信号

### 买入信号（满足3项以上）
1. KDJ金叉（K上穿D）
2. MACD金叉（DIF上穿DEA）
3. MACD柱由负转正
4. RSI < 70（未超买）
5. 5日均线上穿10日均线
6. 股价触及布林带下轨

### 卖出信号（满足3项以上）
1. KDJ死叉（K下穿D）
2. MACD死叉（DIF下穿DEA）
3. MACD柱由正转负
4. RSI > 80（超买）
5. 5日均线下穿10日均线
6. 股价突破布林带上轨

### 止损信号（任一满足）
1. 股价跌破预设止损线
2. 单日跌幅 > 7%
3. 连续3日下跌且跌破20日均线
